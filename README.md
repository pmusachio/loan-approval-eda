## Objetivo
Este projeto analisa uma base de aprovação de empréstimos para identificar quais variáveis estão mais associadas ao `loan_status` e preparar os dados para uso posterior.

O trabalho vai da definição do problema até a etapa de pré-processamento, sem avançar para treinamento de modelos ou avaliação preditiva.

## Dataset
O conjunto de dados utilizado foi o **Loan Approval Classification Dataset**, disponível publicamente na plataforma [Kaggle Dataset Link](https://www.kaggle.com/datasets/taweilo/loan-approval-classification-data?select=loan_data.csv).

## Tecnologias
O notebook foi desenvolvido em **Python** com apoio das bibliotecas:
- **Pandas** para manipulação e análise de dados
- **NumPy** para operações numéricas
- **Matplotlib** e **Seaborn** para visualização
- **Scikit-learn** para divisão treino/teste e pré-processamento
